# Árvores B e B+

## 📚 Sobre o Projeto
Este repositório foi criado para a apresentação sobre **Árvores B e Árvores B+**, estruturas de dados amplamente utilizadas em bancos de dados e sistemas de arquivos devido à sua eficiência em operações de busca, inserção e remoção de dados.

As **Árvores B** são estruturas balanceadas que permitem armazenar grandes quantidades de informações de forma organizada, reduzindo o número de acessos à memória.

Já as **Árvores B+** são uma variação das Árvores B, onde todos os dados ficam armazenados nos nós folha, permitindo buscas sequenciais mais rápidas e melhor desempenho em sistemas de banco de dados.
